Here is my top 5 of the best resources the web has to offer when it comes to learning HTML and CSS and Design.

1. Mozilla Developer Network: The developers from Firefox came to help others by giving web development tutorials, this would be very ideal when beginner to using HTML, CSS and JavaScript. They also give tutorials for advanced users for giving more technical codes and hints and tricks.

2. Codecademy: Would be like a course from university however the user can able to control whenever the user would like to open it and wherever they are. They can give tutorials from beginners to more advanced users. They have also had tutorials for JavaScript, Python and PHP coding after the user has finished tutorials with HTML and CSS.

3. HTML Dog: The website provides tutorials for beginners easy to see the tutorials directly without any flash content like Codecademy. It has tutorials for beginners, intermediate and advanced users.

4. Ruby on Rails: If the user would be interested on building web applications they can able to use this website to have tutorials and see how the web applications can be created.

5.  Code Avengers: This website would provide interactive programming tutorials for the user, they have many languages such as English, Dutch, Spanish, Italian, Russian, Turkish and Portuguese. The tutorials would be consisting of coding for websites like HTML, JavaScript, CSS, Games and applications.
