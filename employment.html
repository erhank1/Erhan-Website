Having looked at the table beneath the differences between the software developer and web developer would be that, the software developer would able to work with applications and programs but the web developer would only specified to work with website development.

Junior Software Developer

The software developer would creative brains behind the computer programming, some of the developers would create apps for the devices like computers or smartphones others would be doing some tasks on the computers. These are the some of duties of the Software Developers:

• Creating new applications

• Maintaining the applications

• Recommend updates to the customers so that they can have latest software running more smoothly and more efficiently

• Analyse the any problem with the code of the software and meeting the specific tasks.

The software developers would be in charge of the software program; they also need to resolve the user requirements when they have unrelated features in the software. Some of the essential Abilities for Software Developers:

• Analysis skills: The software developers would able to analyse to the user needs and able to create software to perform them.

• Communication skills: The software developers must able to give perfect set of instructions when working with other developers and they must able to do the same when communicating with the customers.

• Problem Solving: When they have a problem with the coding of the software they should able to fix it without any difficulties.

The Salary:£21,500 - £48,000

Web Developer

Web developer would be a person creating a webpage for the customers, the customer would able to tell to the web developer the requirements and they can able to create a webpage and meeting those requirements. When the website is complete the web developer must able to test that the website is functioning correctly and without any difficulties.

The essential skills for web developer would be:

• Able to manage time effectively and meeting the deadlines

• Give full attention to the task in hand

• Be creative and give a small touches to make sure that the customer would be pleased

• Being able to know about how the servers and webpages operate

The web developer would be working with these languages:

• HTML(Hyper-Text Markup Language)

• C++

• JavaScript

• Flash

• SQL(Structured Query Language)

Salary: £16,750 - £37,187
